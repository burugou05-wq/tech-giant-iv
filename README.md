# TECH GIANT IV - 開発ドキュメント

このゲームは「1946年から始まり、世界一の巨大テック企業を目指すシミュレーションゲーム」です。
プログラミングに詳しくない方でもゲームの仕組みが理解できるよう、現実の会社や機械に例えて構成を解説します。

---

## 🎮 1. ゲームの動き方（全体の仕組み）

このゲームは、以下の4つの「部署」が連携して動いています。

### 📁 `src/hooks/useGameState.js` ＝ 【金庫 兼 帳簿帳】
今の会社の「所持金」「工場の数」「社員の士気」「今年の年数」など、**すべてのデータ（記憶）を保管している場所**です。
ゲームをセーブしたりロードしたりする場合は、この帳簿の中身を保存することになります。

### 📁 `src/hooks/useGameEngine.js` ＝ 【心臓部（時計）】
ゲーム内で「時間が進む処理」をずっと行っている**ゲームの心臓**です。
現実の時間が1秒経つごとに、「製品を工場で作る」「市場で売って利益を出す」「給料（コスト）を払う」「ライバル会社のシェアを計算する」といった処理を全自動で行い、結果を『帳簿（useGameState）』に書き込みます。

### 📁 `src/hooks/useGameActions.js` ＝ 【社長のコントローラー】
あなたが画面のボタン（例：「マーケティングを強化」「新製品を開発」）を押したときの**アクションの窓口**です。
ボタンが押されると、このファイルにある処理が走り、「お金を減らして、市場の知名度を上げる」といった形で『帳簿』を書き換えます。

### 📁 `src/components/` ＝ 【モニター（画面）】
『帳簿』の数字を読み取って、グラフやボタンとして**画面に表示する**役割です。
デザインやUIを変更したい時は、この中にあるファイル（例：`Market.jsx` や `CorporateFocus.jsx`）をいじります。

---

## 📚 2. ゲームのルールブック

ゲームの「設定」や「歴史のシナリオ」は、以下のフォルダにまとめられています。新しい要素を足したい場合はここを書き換えます。

### 📁 `src/constants/` ＝ 【ルールと設定の辞書】
*   **`market.js`**: ライバル会社（Pineapple, Samstarなど）の強さや登場する年が書かれています。
*   **`corporate.js`**: 先ほど見た「企業方針（ツリー）」のデータが書かれています。「条件を満たしたら解放される」などのルールもここです。
*   **`events.js`**: 「オイルショック」「リーマンショック」などの歴史イベントが何年に発生するか書かれています。

---

## 🤖 3. ちょっと賢い裏方

### 📁 `src/utils/aiSimulation.js` ＝ 【ライバルAIと市場の計算係】
「ライバル会社がいつ新しい商品を出すか？」「プレイヤーの商品と比べて、どっちが市場のシェアを多く奪うか？」といった、**複雑な経済の計算**だけを専門で行うファイルです。『心臓部（useGameEngine）』から毎回計算を依頼されます。

---

## 💡 もし新しいことをやりたくなったら？

*   **「新しいライバル会社を出したい！」**
    👉 `constants/market.js` を書き換えるだけ！
*   **「企業方針のツリーをもっと増やしたい！」**
    👉 `constants/corporate.js` に新しい項目を足すだけ！
*   **「画面のデザイン（色やボタン）を変えたい！」**
    👉 `components/tabs/` の中の該当するファイルをいじります。
*   **「新しいゲームのルール（例：従業員のストライキ機能）を作りたい！」**
    👉 『帳簿（useGameState）』に変数を足し、『心臓（useGameEngine）』に給料計算のルールを書き、『画面（components）』にボタンを足す必要があります（ちょっと大工事になります）。
